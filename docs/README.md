# Capital Quiz

Guess the world's capitals from a single red dot on a globe. No borders, no labels, no hints.

## How to play

1. Open the page and press **Play today's quiz**.
2. A red dot appears somewhere on the globe. That dot marks a capital city.
3. Type the city's name in the box and press **Submit** (or hit Enter).
4. You get 30 seconds per question. The timer bar at the bottom shows how much time is left.
5. After you answer (or time runs out), the country lights up, green if you got it, red if you didn't.ß
6. The quiz is 3 questions long. You can only play once per day.

## Tips

- Spelling doesn't have to be perfect. Small typos and missing accents are accepted.
- Some capitals have multiple common names (e.g. *Kyiv* / *Kiev*); both usually work.
- Today's quiz is the same for everyone, so you can compare scores with friends.

## Resetting

If you want to play again on the same day, add `?reset` to the URL.

Come back tomorrow for a new quiz!
